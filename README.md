# Personal Home Project

- FastAPI, PostgreSQL, Elasticsearch, logstash를 활용한 블로그 개발 프로젝트
- 추후 FastAPI로 작성한 프로그램을 Spring으로 Migration할 계획
- 공부하며 업데이트 되므로 느립니다.

## Data Structure

- Users
- Contents

## Implemented Functions

- Sign up
- Sign in
- Get post list
- View posting
- Add new post
